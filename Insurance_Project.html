Insurance Weekly Branch Report

Project Overview

This project delivers a comprehensive branch report for an insurance company. It utilizes Tableau to visualize key performance metrics, such as Cross-Sell, New, Renewals, Opportunities, Revenue Contributions, and Account Executive performance, enabling data-driven decision-making.

Features

  • Dynamic Cross-Sell, New and Renewal Metrics visualization, showcasing targets vs. achievements.
  • Annual meeting count analysis for evaluating account executive activity.
  • Opportunity distribution across product categories with insightful visuals.
  • Ranking chart displaying top opportunities by revenue.
  • Interactive filters for employee-specific analysis and insights.
