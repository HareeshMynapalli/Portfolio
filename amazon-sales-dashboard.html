Amazon Sales Dashboard

Project Overview

This project presents a comprehensive Amazon Sales Dashboard created using Power BI. It visualizes key metrics such as revenue, profit, units sold, and category performance to help Amazon optimize its sales strategy.

Features

  • Interactive visualization of revenue, profit, and units sold by product category.
  • Monthly and yearly trends in sales and profits.
  • Top-performing products and categories based on key metrics.
  • Dynamic filtering options for region- or category-specific analysis.
